CSS

html태그를 꾸며주는 언어

문법의 구조 - 태그를 선택해서 스타일을 적용한다.

기초문법

CSS에서 태그를 선택하는 방법으로는 크게 3가지가 있다.

  • 태그이름으로 선택하는 방법
  • 클래스 이름으로 선택하는 방법
  • 아이디 이름으로 선택하는 방법

선택문자(선택자){스타일속성:스타일속성값;스타일속성:스타일속성값;...}

선택자는 태그를 선택하는 방법이다.(태그명, 클래스명, 아이디명으로 선택할 수 있다.)

그리고 {} 중괄호 안에는 적용할 스타일들을 속성과 속성값으로 입력한다. 여러개의 속성을 입력해야 하기 때문에 ;(세미콜론)을 사용하여 구분을 해준다.

스타일 입력 방식

  • 내부스타일시트방식 - head태그 안에 style태그 입력 후 스타일을 넣는 방법
  • 인라인스타일방식 - 시작태그에 style 속성을 사용하는 방법
  • 외부스타일시트방식 - 외부의 css문서를 html파일에서 link태그로 연결하는 방법

내부스타일시트의 단점

html파일 안에 스타일코드가 방대하게 들어가게 되면 수정보완하는데에 있어 불편하다.

스타일 코드가 html 파일 안에 있기 때문에 동시에 여러 html 페이지의 스타일 수정이 불가능하다.

인라인스타일방식의 단점

내부스타일시트의 단점처럼 여러 페이지 동시 수정이 불가능하다.

시작태그에 style 속성을 사용하여 스타일을 입력하는 방식이기 때문에 태그의 가독성이 좋지 않다.

외부스타일시트의 단점

여러 페이지의 소스코드를 한 css 파일에 몰아넣는 것이기 때문에 때에 따라서 특정한 태그에게 다른 스타일을 주고싶을 때 신경을 써야한다.
