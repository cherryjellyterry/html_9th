flex

플렉스박스는 크게 부모태그(플렉스컨테이너) 자식태그(플렉스아이템) 두 부분으로 나뉘며 각각의 부분마다 사용할 수 있는 css 속성이 다르다.

플렉스 아이템들은 플렉스 컨테이너의 자식태그들을 말하며, 기본적으로는 플렉스 컨테이너 안에서 컨테이너의 메인 축을 따라 나란히 배치되는 특징을 갖는다.

교차축은 메인축의 방향에 따라서 자동으로 바뀐다.

flex-container(대장)에 지정하는 속성

  • flex-direction : 메인축의 방향을 바꿔주는 속성

    • row(기본값) : 좌->우
    • row-reverse : 우->좌
    • column : 위->아래
    • column-reverse : 아래->위

    플렉스 아이템들은 무조건 메인축의 시작지점부터 나란히 배치된다.

  • justify-content : 메인축을 따라 아이템을 정렬하는 방법

    • flex-start(기본값) : 아이템들을 메인축 시작지점부터 정렬
    • flex-end : 메인축 마지막 지점에 정렬
    • center : 메인축 중앙 지점에 정렬
    • space-between : 메인축의 시작과 마지막 지점에 아이템을 맞춘 후 아이템들 간의 사이 여백을 동일하게 지정
    • space-around : 아이템들의 사이여백을 동일하게 지정하여 정렬

  • align-items : 교차축을 따라 아이템을 정렬하는 방법

    • stretch(기본값) : 교차축 길이만큼 아이템 길이를 늘림
    • flex-start : 교차축의 시작지점에 아이템 정렬
    • flex-end : 교차축의 마지막지점에 아이템 정렬
    • center : 교차축의 중앙지점에 아이템 정렬

  • flex-wrap : flex-item들의 줄바꿈을 설정하는 속성

    • no-wrap(기본값) : 줄바꿈 허용 안함
    • wrap : 아이템들의 줄바꿈 허용

  • align-content

    • flex-container에서 아이템들의 줄바꿈을 허용했을 때 container 전체 높이를 수직축으로 하여 아이템 그룹 전체를 정렬할 때 사용하는 속성

flex-item(쫄병)들에게 지정하는 속성

  • flex-basis

    • width처럼 아이템의 가로 사이즈를 지정할 수 있다. 기본값은 auto
    • 플렉스 아이템들이 점유하는 기본 크기를 설정

  • flex-shrink

    • 아이템의 크기가 줄어드는 비율
    • 플렉스 컨테이너의 너비가 줄어들어서 아이템들이 설정된 너비보다 작아져야 할 때, 줄어드는 비율을 말한다.(기본값 :1) - 아이템들이 줄바꿈이 허용되지 않는 상황에서 유효하다.
    • 기본값은 1이며 컨테이너 공간이 줄어들면 아이템들의 너비가 같이 같은 비율로 줄어든다.

  • flex-grow

    • 플렉스 컨테이너의 공간이 여유가 있어서 남을 때, 남은 공간을 아이템들이 나눠갖는 비율
    • 기본값은 0 이며 이 경우에는 남은 공간이 있어도 아이템의 크기는 커지려고 하지 않는다.

  • order

    • 기본값 : 0.
      아이템의 보여지는 순서를 바꿀 때 사용
    • order 값이 클 수 록 나중에 배치됨

  • align-self

    • 선택한 아이템의 수직축 정렬
