배경 - background

배경은 컨텐츠가 없거나 높이가 없으면 배경표현이 되지 않는다.

배경스타일이 들어가는 공간은 태그의 가장 밑바닥이다.(컨텐트박스보다 아래에 깔린다.)

background-color: blue; color: white; padding: 20px; height: 300px; width: 300px; box-sizing: border-box;

background-color

배경색상 입력

태그의 공간(테두리 안쪽 모든 공간)에 배경색을 나타낼 때 사용하는 css속성

height: 80vh; margin: 100px 0; background-color: bisque; background-image: url(../img/jason-leung-AydnLKqj4Ug-unsplash.jpg); background-repeat: no-repeat; background-position: 100px 200px;

background-image

배경 이미지 입력 - url() 도구 사용

배경이미지는 입력하면 자동으로 바둑판으로 반복된다.

배경이미지가 반복되는 속성

repeat(기본값-가로 세로로 반복)

no-repeat(반복 안함)

repeat-x(가로로 반복)

repeat-y(세로로 반복)

background-position

배경이미지의 위치를 변경할 때 사용하는 속성

background-position: x좌표, y좌표

키워드방식, %방식, 픽셀방식

height: 100vh; background-color: aquamarine; background-image: url(../img/jason-leung-AydnLKqj4Ug-unsplash.jpg); background-repeat: no-repeat; /* background-size: 300px auto; */ /* 가로 세로 */ /* background-size: auto 300px; */ /* background-size: 100% auto; background-size: auto 100%; */ background-size: cover; /* 화면 사이즈에 관계없이 자동으로 늘어나거나 줄어들면서 전체화면을 감쌈. 배경이미지가 잘려서 안보일 수도 있음. */ background-size: contain; /* 이미지가 태그의 공간 안에 잘리지 않는 선에서 최대 크기로 들어가서 채우는 방식 */ background-position: center center; background: aquamarine url(../img/jason-leung-AydnLKqj4Ug-unsplash.jpg) no-repeat 0 0 /cover; /* 백그라운드 사이즈 쓸 때에는 앞에 / 써야함. */ /* 순서도 중요함 */ /* 대신 축약형으로 쓰면 다음에 background: red; 만 적용해도 다 없어지고 빨간배경됨. */

cover

배경이미지 크기를 이미지가 잘려나가더라도 태그의 공간을 가득 메우는 방식

contain

배경이미지가 잘려나가지 않는 선에서 배경이미지의 크기를 최대한 키워서 넣는 방식

축약형 표현

background: 배경색 이미지 반복속성 좌표 /사이즈 순서로 씀(순서 중요! 외울것)

blur-contents
