a태그(링크태그)

사용자를 의도한 목적지로 보내주는 태그

a태그는 anchor(닻)라는 뜻으로 사용자를 원하는 목적지로 이동시켜줄 때 사용한다.

a태그는 href 속성 없이 사용될 수 없다. 반드시 필요한 속성이다. href 속성에 들어가는 속성값은 목적지로 갈 수 있는 경로를 입력하면 된다.

a태그는 마치 강조태그처럼 필요한 단어나 문장을 강조하듯 링크가 필요한 곳에 자유롭게 사용하면 된다.

a태그에서 target 속성의 기본값은 _self로 설정되어 있어서 사용자가 링크 태그를 클릭했을 때, 사용자가 보고 있는 현재 탭에서 이동된다. 이 값은 기본값이므로 target 속성을 코드에 쓰지 않아도 a태그에 항시 적용되는 값이다.

target 속성값을 _blank로 설정하면 기본적으로 설정되어 있던 _self 값은 무시되고 현재 탭에서 열리지 않으며 새 탭에서 사용자가 클릭한 링크의 목적지 페이지가 열리게 된다.

경로 입력 방법

절대경로/상대경로 2가지가 있다.
절대경로 방법은 인터넷 주소를 복사해와서 그대로 사용하는 방법이며, 주소 표시창에 있는 주소를 그대로 복사해서 사용하면 된다.

상대경로

현재 파일에서 목적지 파일까지의 경로를 표현하는 방법

그래서 동일한 목적지 파일에 도달한다는 가정 하에 선택된 현재 파일의 위치에 따라서 경로는 달라질 수 있다.

. -> 현재 파일의 위치를 불러온다.

/ -> 폴더를 연다.

.. -> 상위 폴더(부모폴더)의 위치를 불러온다.
