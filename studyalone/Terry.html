강조태그 및 특수문자태그

b태그, strong태그는 강조태그임. 자동으로 줄바꿈되지 않기 때문에 아무데에나 써도 됨.

강조하는방법에는 이 밖에도 이탤릭체로도 구현할 수 있는데, 이 때에는 em태그나 i태그를 사용한다.

속성 태그

기본 형태는 <태그명 속성명="속성값">. 속성은 순서가 중요하지 않음.

span태그는 글자 하나하나에 속성을 부여할 때 사용함.

같은 기능끼리는 한 태그 안에 넣을 수 있지만, 다른 기능은 따로 부여해야 함. 같은 기능이라면 개수나 순서에 상관없이 작성할 수 있음. 중간에 띄어쓰기를 통해 구분함.

style="font-size:원하는 글자크기"는 폰트 사이즈를 바꾸는 속성임. style="color:#ffffff"는 색깔속성임.

a태그와 속성

a태그는 링크를 거는 태그임. a href="주소" 의 형태를 가지며 시작태그와 마침태그 사이에는 링크에 부여할 이름을 적음.

href 없이는 사용할 수 없고, 강조태그처럼 아무데서나 사용할 수 있음.

a태그에서 target속성(창이 어디서 열릴건지) 기본값은 self로, 현재 탭에서 링크가 열리는 것임. 이것을 blank로 설정하면 새탭에서 링크가 열리게 됨.

title 속성은 클릭하기 전에 커서를 위에 올려두면 뜨는 안내창을 설정하는 속성임.

블랭크 속성을 적용한 네이버링크 기본값(self)속성을 적용한 네이버 링크 타이틀 속성을 적용한 네이버링크(커서를 올려보세요)

이미지태그

img태그에 이미지파일을 연결해서 보여줄 수 있음. 대부분의 형태를 지원하지만, 브라우저마다 지원하는 이미지 확장자가 다르기 때문에 주로 사용하는 브라우저에 맞춰서 사용해야 함.

img태그에도 style속성을 사용할 수 있음. height 같은걸로 보이는 크기 줄이기 가능

표 태그

table태그를 만들면 보이지 않는 표 공간이 만들어짐. 그 안에 컨텐츠를 입력하는 순서는 왼쪽에서 오른쪽으로. 첫번째 줄 태그를 먼저 모두 입력한 뒤 두번째 줄로 넘어가야 함.

셀 병합은 표를 다 만든 후 해야함. 병합이 시작되는 셀(가장 왼쪽, 가장 위쪽)을 찾아서 가로병합(colspan)/세로병합(rowspan) 속성을 사용. 이 때, 병합이 되어져서 사라지는 셀은 태그를 지워야 함.

  • 자식태그 caption : 표제목
  • 자식태그 tr : 가로 행 만들기
    • tr의 자식태그 td : 가로행 안의 셀태그
    • tr의 자식태그 th : 가로행 안의 셀태그 중 분류기준(굵게, 가운데정렬)

table border="1" : 표의 선 굵기를 1로 설정하는 속성
